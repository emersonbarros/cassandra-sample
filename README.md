# Cassandra Sample

This is a sample of Cassandra using mapping feature

### Prerequisites

```
 Eclipse
 Maven
 Docker
```

## Running the tests

```
 mvn eclipse:eclipse

 ./docker/start-cassandra
```

## Built With

* [Eclipse](http://www.eclipse.org/) - IDE
* [Maven](https://maven.apache.org/) - Dependency Management
* [Docker](https://www.docker.com/) - Docker
